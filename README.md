# 精准射手 - CS2 网页瞄准训练器

> 一个在浏览器中运行的、兼具立体空间感与纯粹定位反馈的 FPS 热身工具。

## 🚀 如何开始

1. 访问网址：[https://cs2-aim-trainer-zjkm.vercel.app](https://cs2-aim-trainer-zjkm.vercel.app)
2. 点击网页中央的 **“点击开始”** 区域。
3. 浏览器弹出权限请求，点击 **“允许”** 锁定鼠标。
4. 移动鼠标控制准星，**点击左键** 射击气球。

## 🎮 核心玩法

- **模式 A（单目标）**：练习快速定位与单发精度。
- **模式 C（多目标）**：练习连续拉枪与压力应对。
- **动态难度**：得分越高，气球越小越远，持续挑战极限。
- **60 秒限时赛**：时间结束自动结算，展示命中率与平均反应时间。

## 📊 数据反馈

每局结束后展示：
- 命中率
- 平均反应时间（毫秒）
- 最长连击数

## ❓ 常见问题

**Q: 为什么鼠标不见了？**  
A: 这是为了 FPS 游戏手感设计的**指针锁定**。按键盘左上角的 `ESC` 键即可恢复鼠标。

**Q: 为什么没有声音？**  
A: 浏览器安全策略限制，首次点击画面后声音才会激活。

**Q: 手机可以玩吗？**  
A: 本项目专为**桌面端鼠标**设计，手机浏览器可能无法正常锁定指针，建议在电脑上使用 Chrome 或 Edge 浏览器访问。
